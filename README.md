# Hyprland-setup

A simple setup script for installing and configuring **Hyprland (Wayland compositor)** with a ready-to-use desktop environment.

This project is mainly intended for:

## Installed packages

The setup installs (at minimum):

- **hyprland** – Wayland compositor  
- **waybar** – status bar  
- **rofi** – application launcher  
- **alacritty** – terminal emulator  
- **swww** – wallpaper daemon  
- **grim** – screenshot utility  
- **network-manager-applet** – network tray icon  
- **nautilus** – file manager  
- **otf-font-awesome** – icons for Waybar  

Additional packages can be added easily in the script.

## Keybinds

Super + T -> Terminal
Super + W → close window
Super + M → exit hyprland
Super + E → file manager
Super + V → floating
Super + Space → menu (rofi)
Super + P → pseudo-tiling
Super + J → split
